# Grapple Platformer
### Author: Harmehel Singh
### Mr. Polacco 

This is a WIP swinging-grapple platformer, made for my AP CS A final project.

# To Compile
```
javac -d build Main/*.java Listener/*.java Class/*.java 
java -cp build Main.GrapplePlatformer
```

# Create .jar File
```
jar cfe GrapplePlatformer.jar GrapplePlatformer *.class
```
